Getting Started

Installation

Install using npm, yarn or pnpm:

npm install typeorm-extensions

Importing

Import the entire library:

import 'typeorm-extensions';

Import the specific extension:

import 'typeorm-extensions/dist/extensions/typed/typed-where/typed-where.extension';

API Reference

API documentation is available here

Usage

Pagination

query.applyPaginationFilter({ page: 1, pageSize: 10 }, { useTakeAndSkip: true })

Typed Where

query.whereTyped(user => user.name, 'ILIKE :search', { search: value })

Typed Order By

query.orderByTyped(user => user.name, 'ASC')

Typed Select

query.selectTyped(user => user.name)

Typed Join

query.leftJoinTyped(user => user.address, 'address')


For other examples please visit the Examples page.
