  • Pagination

Description

applyPaginationFilter is a method that applies pagination to a query.

Signature

applyPaginationFilter(paginationFilter: PaginationFilter, paginationOptions?: ApplyPaginationOptions): void

Arguments

  • paginationFilter: PaginationFilter

    The pagination options to apply to the query.

    Properties

    • page: number

      The page number to retrieve.

    • pageSize: number

      The number of items per page.

  • (optional) paginationOptions?: ApplyPaginationOptions

    Additional options to apply to the pagination filter.

    Properties

    • useTakeAndSkip?: boolean

      Whether to use the take and skip methods instead of limit and offset (defaults to false).

Usage

query.applyPaginationFilter({ page: 1, pageSize: 10 }, { useTakeAndSkip: true })
