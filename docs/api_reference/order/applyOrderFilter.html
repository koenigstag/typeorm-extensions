Description

applyOrderFilter is a method that applies order to a query.

Signature

applyOrderFilter(orderFilter: OrderFilter, orderOptions?: ApplyOrderOptions): void

Arguments

  • orderFilter: OrderFilter

    The order options to apply to the query.

    Properties

    • orderBy: Array< OrderParam >

      Order configurations.

  • (optional) orderOptions?: ApplyOrderOptions

    Additional options to apply to the order filter.

    Properties

    • useDoubleQuotes?: boolean

      Whether to wrap the order field with double qoutes (defaults to false).

    • ...TBD...

Usage

query.applyOrderFilter({ orderBy: [{ field: 'name', direction: 'asc', nulls: 'last' }] }, { useDoubleQuotes: true })
